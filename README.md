# TORRES BARBER CLUB

## Sistema Web Integrado para Gestão de Barbearia

---

### RESUMO

O Projeto Integrado Multidisciplinar (PIM) é um recurso metodológico idealizado pela Universidade Paulista (UNIP) que tem como finalidade incentivar a integração interdisciplinar dos conhecimentos adquiridos ao longo do curso de Análise e Desenvolvimento de Sistemas, por meio da realização de um trabalho acadêmico que possibilite o desenvolvimento teórico-prático dos alunos. O PIM III, componente curricular obrigatório, propicia ao estudante a oportunidade de aplicar habilidades técnicas e sociais visando à resolução de problemas reais ou simulados em um contexto empresarial.

O presente trabalho teve como objetivo o desenvolvimento de um Sistema Web Integrado para a barbearia fictícia **Torres Barber Club**, estabelecimento que atua no segmento de serviços de beleza masculina e que atende a diferentes perfis de clientes, incluindo pessoas com necessidades especiais. O sistema objetivou otimizar o agendamento de serviços, o controle de estoque, o gerenciamento do histórico dos clientes e promover a inclusão digital por meio do suporte à Língua Brasileira de Sinais (LIBRAS), buscando atender às demandas operacionais e sociais do negócio.

Como metodologia, foi adotada a Engenharia de Software Ágil, estruturando o projeto em sprints, com definição e priorização de backlog, e criteriosa avaliação das entregas. Aliou-se ainda a essa abordagem a modelagem de dados utilizando bancos de dados relacionais e NoSQL, a programação orientada a objetos com C#, o desenvolvimento de interfaces web responsivas utilizando HTML5, CSS3 e JavaScript, além da aplicação de técnicas de UX/UI Design, análise de dados e machine learning. O processo de desenvolvimento contemplou também competências de comunicação, liderança, negociação e inclusão social para promover um trabalho colaborativo e humanizado.

**Palavras-chave:** Sistema Web. Gestão de Barbearia. C#. SQL. UX/UI. LIBRAS. PIM.

---

### 1. INTRODUÇÃO

#### 1.1 Contextualização

A barbearia **Torres Barber Club** é uma empresa fictícia criada para o contexto deste projeto acadêmico. O estabelecimento atua no segmento de serviços de beleza masculina, oferecendo cortes de cabelo, barba modelada, tratamentos capilares e produtos premium para cuidados pessoais.

#### 1.2 Problema Identificado

Actualmente, muitas barbearias de pequeno porte enfrentam dificuldades na gestão operacional, utilizandoplanilhas de Excel e mensagens de WhatsApp para controlar agendamentos. Essa prática gera:
- Conflitos de horários
- Perda de clientes por falta de atendimento personalizado
- Dificuldade no controle financeiro
- Ausência de histórico de preferências dos clientes

#### 1.3 Objetivo Geral

Desenvolver um sistema web integrado que automate os processos de agendamento, gestão de clientes, controle de estoque e relatórios financeiros, promovendo também a inclusão digital com suporte a LIBRAS.

#### 1.4 Objetivos Específicos

- Desenvolver um módulo de agendamento online
- Implementar cadastro e histórico de clientes
- Criar dashboard para gestão financeira
- Implementar controle de estoque de produtos
- Adicionar recursos de acessibilidade (LIBRAS)
